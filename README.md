> 캐주얼 모바일 리듬 게임, 네 알바 아냐



## 기획

아르바이트를 하여 돈을 모아 자취방을 커스터마이징 하는 캐주얼 게임



## 기능 구현

- C#과 Unity를 이용한 게임 구현
- 자취방 커스터마이징 기능
- Google Firebase Realtime DB를 이용한 랭킹 시스템

## 커밋 로그 규칙

`feat: __  ` 기능추가

`fix: __ ` 수정

`add: __ `파일추가

`refactor: __ ` 코드 리팩토링

`ui: __` - ui 추가 및 변경

`chore` 기타 잡다한 일 

모두 동사로 시작할 것



##### **예시**

`feat: pause`

`fix: object respawn error`
